規則說明
regions.name = units.region 集團總覽的地區卡片靠這個對應；若不一致，該地區顯示「0 單位」
training.participants 姓名 = talents.name 課程安排用姓名字串比對 talents，找不到則無法顯示 ⭐ 種子標記
cases.stage 只能填 poc 或 completed 填其他值 tab 篩選不會計入，stageTag 顯示灰色
布林欄位（needSupport / isReplicable / isSeed） 填 TRUE / FALSE（大寫），或 true / 是 皆可
陣列欄位（pocItems / toolsUsed / skills 等） 以英文逗號分隔，sheets.js 會自動 split：PaddleOCR, Python, Cloud Run
maturityLevel 格式 只能填 L0–L5，大寫 L + 數字；填錯會 fallback 到 L0
